Installation Guide

Get started in just a few minutes

1

Download the Skill

Download the blender-mcp-feedback folder or the .skill package file.

2

Add to Claude.ai

Settings → Skills → "Add Custom Skill" → Upload the folder. Instantly available in all chats!

3

Load in Blender

Copy the blender_quick_template.py into Blender Script Editor (Alt+Tab) and run it.

exec(open('/path/to/blender_quick_template.py').read())
4

Capture Viewport

Initialize and capture your scene state in just two commands:

mcp = QuickBlenderMCP() state = mcp.capture_and_export()
5

Send to Claude

Copy the printed JSON scene state and send it to Claude with context about your project.

Please review this Blender viewport and suggest improvements.
6

Execute Commands

Claude returns MCP commands. Execute them to automatically refine your render:

mcp.apply_changes(commands)
